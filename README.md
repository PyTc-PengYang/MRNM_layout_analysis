

此存储库包含 MRNM（多模态特征和关系预测网络）的实现，这是一种用于历史文档布局分析的模型。MRNM 将多模态特征（视觉、文本和空间）与关系预测网络相结合，以检测布局元素并预测阅读顺序。

模型架构 MRNM 由三个主要部分组成：
多模态网络 ：提取并融合视觉、文本和空间特征
区域检测 ：执行布局元素检测和分类
关系预测网络 ：使用图神经网络预测读取顺序
